# KsanaDock GameMaker

[简体中文](./README.zh-CN.md) | [English](./README.en.md)

AI-powered game development assistant for the Godot Editor.

`KsanaDock GameMaker` currently includes:

- `KsanaDock`: chat, login, profile, context capture, and terminal UI inside Godot
- `KsanaDock Bridge`: local WebSocket bridge between Godot Editor and the agent service
- `Agent Service`: a Node.js-based service that handles agent loop, tools, and skill loading
